title: Android
description: Install Coords on Android via F-Droid or Obtainium
---

Coords for Android is distributed through our own F-Droid repository. You can install it using F-Droid or Obtainium.

## F-Droid

Add [our repository](https://coord.is/fdroid/repo/) to F-Droid:

1. Open **Settings** → **Repositories**
2. Tap **+** to add a new repository
3. Scan the QR code at [coord.is/fdroid/repo](https://coord.is/fdroid/repo/) or enter the URL manually:
   ```
   https://coord.is/fdroid/repo/
   ```
4. Tap **Add**

Then search for **Coords** and install.

## Obtainium

[Obtainium](https://github.com/ImranR98/Obtainium) can pull directly from our F-Droid repo:

1. Tap **Add App**
2. Enter the repository URL:
   ```
   https://coord.is/fdroid/repo/
   ```
3. Set **Override Source** to `F-Droid Third-Party Repo`
4. Set **App ID** to:
   ```
   sh.bentley.transponder
   ```
5. Tap **Add**

Obtainium will notify you when updates are available.
